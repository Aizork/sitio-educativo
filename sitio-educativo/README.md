# Sitio Educativo Proyecto II

Un sitio web educativo interactivo con visualización de contenido y funcionalidad de pruebas básicas.

## Desarrollado por
- Jessica Nayeli Joselinne Cuellar Rivera 224065839
- Aldo Guerrero Ortega 222353187

## Características
- Navegación de cursos por categorías
- Contenido educativo interactivo
- Sistema de cuestionarios y evaluación
- Seguimiento de progreso del estudiante
- Interfaz moderna y responsiva
- Soporte para modo oscuro

## Estructura del Proyecto
- `/client`: Componentes y páginas de React para la interfaz de usuario
- `/server`: Código del servidor Express
- `/shared`: Esquemas y tipos compartidos entre cliente y servidor

## Tecnologías utilizadas
- React
- TypeScript
- Tailwind CSS
- Express.js
- Node.js
- Wouter para enrutamiento
- TanStack Query para gestión de estado
- Zod para validación de esquemas
- Drizzle ORM para acceso a datos

## Instalación y ejecución

### Requisitos previos
- Node.js 18 o superior
- npm 8 o superior

### Pasos para ejecutar localmente
1. Clonar el repositorio
```bash
git clone https://github.com/tu-usuario/sitio-educativo-proyecto-ii.git
cd sitio-educativo-proyecto-ii
```

2. Instalar dependencias
```bash
npm install
```

3. Iniciar el servidor de desarrollo
```bash
npm run dev
```

4. Abrir en el navegador: `http://localhost:5000`
